## 启动
- 参数
    - -user: 访问网页的用户名
    - -pwd: 访问网页的密码
    - -js-path: jquery库的路径
    - -host: http服务的监听ip
    - -port: http服务的监听端口
    - -cfg: 配置进程的配置文件路径

- 完整命令
    - ./watchdog [-user [name]] [-pwd [pwd]] [-js-path [jspath]] [-host [host]] [-port [port]] [-cfg [cfgpath]]

## 网页查看
- 浏览器输入:
    - http://ip:port/index

- 如:
    - http://127.0.0.1:51000/index
